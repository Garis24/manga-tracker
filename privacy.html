Politique de confidentialité

Dernière mise à jour : avril 2026

Manga Tracker est une extension Chrome développée par Tiudge. Cette politique explique quelles données sont traitées, pourquoi elles le sont, et comment elles sont stockées pour assurer le suivi de lecture manga et la synchronisation optionnelle entre appareils.

1. Objectif du service

Manga Tracker a pour objectif de sauvegarder votre progression de lecture manga automatiquement, site après site, afin que vous puissiez reprendre exactement au bon chapitre.

Le projet vise une compatibilité aussi large que possible avec les sites de lecture manga, avec repérage visuel des chapitres lus, bibliothèque locale, notes, statuts de lecture, synchronisation Google Drive optionnelle, compatibilité mobile via Kiwi Browser et notification de mise à jour.

2. Données traitées

Manga Tracker traite uniquement les données nécessaires à son fonctionnement :

  • les URL des pages de lecture visitées sur les sites compatibles, pour détecter le manga et le chapitre ;
  • les titres, slugs, domaines et identifiants utiles pour reconnaître un manga ;
  • les numéros de chapitres marqués comme lus ;
  • les statuts de lecture, notes personnelles et préférences d’affichage ;
  • les informations minimales nécessaires à la synchronisation Google Drive si vous activez cette option.

3. Stockage des données

Par défaut, les données sont stockées localement sur votre appareil via l’API chrome.storage.local.

Si vous activez la synchronisation Google Drive, les données sont également sauvegardées dans le dossier AppData de votre Google Drive personnel. Ce dossier est masqué dans l’interface Google Drive et reste dédié aux données techniques de l’application.

4. Partage des données

Manga Tracker ne vend pas vos données et ne les partage pas avec des tiers à des fins publicitaires, analytiques ou commerciales.

Aucune donnée de lecture n’est transmise à l’auteur de l’extension. Lorsque la synchronisation Google Drive est activée, les données sont envoyées uniquement vers le compte Google de l’utilisateur, dans son propre espace Google Drive.

5. Google Drive (optionnel)

La synchronisation Google Drive est entièrement optionnelle. Si vous l’activez :

  • l’extension demande uniquement l’accès nécessaire au stockage dans drive.appdata ;
  • aucun autre fichier personnel de votre Google Drive n’est utilisé par l’extension ;
  • vous pouvez révoquer cet accès à tout moment depuis votre compte Google.

6. Permissions Chrome utilisées

L’extension utilise certaines permissions Chrome strictement pour fournir ses fonctions principales.

  • storage : sauvegarder localement la bibliothèque, les chapitres lus, les notes, les statuts et les réglages.
  • identity : permettre la connexion Google pour activer la synchronisation Google Drive de manière optionnelle.
  • tabs : détecter l’URL de l’onglet actif et reconnaître la page manga ou chapitre consultée.
  • activeTab : interagir avec la page en cours quand l’utilisateur ouvre le popup ou déclenche une action liée au suivi.
  • alarms : planifier certaines tâches internes liées à la synchronisation, à la maintenance légère ou à la vérification de disponibilité d’une mise à jour.
  • notifications : informer l’utilisateur lorsqu’une action utile doit être signalée, comme une mise à jour disponible.
  • host_permissions : permettre au script de contenu de fonctionner sur les sites de lecture compatibles afin d’identifier les chapitres lus et d’afficher les repères visuels directement sur les pages.

7. Code distant

Manga Tracker n’exécute pas de code JavaScript distant. Le code nécessaire à son fonctionnement est inclus dans le package de l’extension.

Les services externes éventuellement appelés, comme Google Drive pour la synchronisation, sont utilisés uniquement pour l’échange de données et non pour charger ou exécuter du code distant.

8. Données des mineurs

Cette extension n’est pas destinée aux enfants de moins de 13 ans. Nous ne collectons pas sciemment de données personnelles de mineurs.

9. Modifications

Cette politique peut être mise à jour pour refléter l’évolution de l’extension, de ses permissions ou de ses fonctionnalités. La date de dernière mise à jour est indiquée en haut de cette page.

10. Contact

Pour toute question relative à la confidentialité, vous pouvez consulter le site du projet garis24.github.io/manga-tracker ou le dépôt GitHub github.com/Garis24/manga-tracker.
