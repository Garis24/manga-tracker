Politique de confidentialité

Dernière mise à jour : avril 2026

Manga Tracker est une extension Chrome développée par Tiudge. Cette politique explique quelles données sont collectées, comment elles sont utilisées et stockées.

1. Données collectées

Manga Tracker collecte uniquement les données nécessaires à son fonctionnement :

  • Les URLs des pages de lecture visitées (pour détecter le manga et le numéro de chapitre)
  • Les titres et slugs des mangas lus
  • Les numéros de chapitres marqués comme lus
  • Les statuts et notes personnelles associés à chaque manga

2. Stockage des données

Toutes les données sont stockées localement sur votre appareil via l'API chrome.storage.local.

Si vous activez la synchronisation Google Drive (optionnelle), les données sont également sauvegardées dans le dossier AppData de votre Google Drive personnel, qui est invisible dans votre Drive et accessible uniquement par cette extension.

3. Partage des données

Manga Tracker ne partage aucune donnée avec des tiers, des serveurs externes, ou des services d'analyse. Aucune donnée n'est envoyée à l'auteur de l'extension.

4. Google Drive (optionnel)

La synchronisation Google Drive est entièrement optionnelle. Si vous l'activez :

  • L'extension demande uniquement la permission drive.appdata (dossier caché, non visible dans votre Drive)
  • Aucun autre fichier de votre Drive n'est accessible
  • Vous pouvez révoquer l'accès à tout moment depuis les paramètres de votre compte Google

5. Permissions Chrome

L'extension utilise les permissions suivantes :

  • storage : pour sauvegarder vos données localement
  • identity : pour l'authentification Google Drive (optionnel)
  • tabs : pour détecter l'URL de l'onglet actif
  • activeTab : pour interagir avec la page en cours
  • host_permissions (<all_urls>) : pour fonctionner sur tous les sites de scans

6. Données des mineurs

Cette extension n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons pas sciemment de données personnelles de mineurs.

7. Modifications

Cette politique peut être mise à jour. La date de dernière modification est indiquée en haut de cette page. L'utilisation continue de l'extension vaut acceptation des modifications.

8. Contact

Pour toute question relative à la confidentialité, vous pouvez contacter le développeur via GitHub : github.com/Tiudge/manga-tracker
