Manga Tracker

Extension Chrome pensée pour suivre automatiquement votre progression manga. Le projet se veut compatible avec n’importe quel site de lecture manga, avec synchronisation Google Drive et prise en charge mobile testée sur Kiwi Browser.

Gratuit & Open-source 📚 Compatible avec le plus de sites possible ☁️ Google Drive Sync 📱 Testé sur Kiwi Browser

Aperçu de l'extension

Votre lecture sauvegardée,
site après site

Que vous changiez de site, de manga ou d'appareil, Manga Tracker garde votre progression à jour automatiquement. Vous reprenez votre lecture exactement au bon chapitre, sans avoir à chercher où vous en étiez.

Le projet vise une compatibilité aussi large que possible avec les sites de lecture manga, pour offrir un suivi simple et universel.

🔗 Installer l'extension

Disponible en chargement local via le mode développeur Chrome.

Télécharger l'extension
  1. Téléchargez le fichier ZIP.
  2. Extrayez le dossier manga-tracker-extension à l’endroit où vous voulez le conserver.
  3. Ouvrez Chrome puis allez sur Copié !
  4. Activez le Mode développeur en haut à droite.
  5. Cliquez sur Charger l’extension non empaquetée.
  6. Sélectionnez le dossier manga-tracker-extension.
  7. Épinglez l’extension, puis profitez 🎉
Après installation, une alerte de nouvelle version peut aider à savoir quand mettre à jour l’extension si vous l’utilisez encore en mode développeur.
