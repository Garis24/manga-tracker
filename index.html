Manga Tracker

Extension de suivi manga compatible avec Chrome, Opera GX, Firefox et Kiwi Browser sur Android. Sauvegarde automatique, repérage des chapitres lus et synchronisation Google Drive pour reprendre votre lecture sans perdre votre progression.

Gratuit & Open-source 🖥️ Chrome / Opera GX / Firefox 📱 Kiwi Browser (Android) ☁️ Google Drive Sync

Aperçu

Votre progression manga, retrouvée instantanément

Manga Tracker enregistre automatiquement vos chapitres lus et vous aide à reprendre votre lecture exactement là où vous vous êtes arrêté, même en changeant de site ou d’appareil.

Le projet vise une compatibilité large avec les sites de lecture manga, sur desktop comme sur mobile.

Fonctionnalités

🟢

Repérage automatique

Les chapitres déjà lus sont identifiés visuellement pour reprendre votre lecture en un coup d’œil.

📖

Sauvegarde auto

Votre progression est enregistrée dès l’ouverture d’un chapitre, sans action manuelle.

☁️

Sync Google Drive

Votre bibliothèque et votre avancement restent synchronisés entre vos appareils.

🌐

Compatibilité large

Le projet vise un suivi aussi universel que possible sur les sites de lecture manga.

📱

Mobile compatible

L’expérience mobile a été testée sur Kiwi Browser pour Android.

⬆️

Mises à jour

Le projet évolue régulièrement pour améliorer la compatibilité, la synchro et l’expérience utilisateur.

Installer dès maintenant

Installer depuis le Chrome Web Store
